# dotfiles

## Requirements
```
pacman -S stow git
````

## Installing
```
git clone https://github.com/myzb/dotfiles.git
cd dotfiles
stow -t ~/ .
```

## Setup

### zsh
```
pacman -S zsh zoxide fzf man
```

### neovim
```
pacman -S neovim ttf-jetbrains-mono-nerd fd ripgrep npm
```

### tmux
```
pacman -S tmux inetutils
```

### alacritty
```
pacman - S alacritty
```
